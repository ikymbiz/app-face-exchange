# LiveFace AI

## 概要・説明
LiveFace AIは、Webブラウザ上で動作するリアルタイムの顔変換・トラッキングアプリケーションです。ユーザーの顔をWebカメラで認識し、アップロードした任意の顔写真やイラストをリアルタイムで自分の顔に重ね合わせて表示（オーバーレイ）することができます。

GoogleのMediaPipe Face Meshを活用した高精度な顔認識により、アプリのインストールや複雑な設定は一切不要。ブラウザを開くだけで、手軽にVTuberのような体験や顔の匿名化、おもしろ動画の撮影を楽しむことができます。

## 主な機能一覧
- **リアルタイム顔追跡・オーバーレイ**: Webカメラの映像から顔の向き、傾き、位置を瞬時に検知し、アップロードした画像を自然に合成します。
- **簡易表情追従（リップシンク）**: 口の開閉に合わせてオーバーレイ画像の縦幅を動的に変化させ、擬似的な表情の連動を実現します。
- **AI顔自動切り抜き**: 画像をアップロードする際、AIが自動で顔部分のランドマークを検知し、適切なサイズと丸型に切り抜きます（背景の簡易除去）。
- **動画録画・写真撮影**: 
  - 写真撮影：現在の画面を静止画（JPEG）として即座にダウンロード。
  - 動画録画：合成された映像を動画（MP4/WebM）として録画。マイク音声の同時収録にも対応しています。
- **履歴・データ管理機能**: 
  - 直近5件の顔画像を履歴としてブラウザ（IndexedDB）に自動保存し、タップするだけで過去の画像をすぐに復元できます。
  - 設定や履歴データを含めたJSON形式でのバックアップ（エクスポート/インポート）に対応。
- **モバイルファースト＆PWA対応**: スマートフォンでの利用に最適化されたUI。画面の自動スリープを防ぐWake Lock機能も備えています。
- **SNS共有機能**: 撮影した画像をワンタップでOS標準のシェア機能（Web Share API）を使ってSNS等へ共有できます。

## 使用技術・ライブラリ
- **フロントエンド / UI**: HTML5, JavaScript (Vanilla), Tailwind CSS (CDN), FontAwesome 6
- **AI・機械学習 (顔認識)**: MediaPipe Face Mesh, MediaPipe Camera Utils
- **Web API**: 
  - **Canvas API**: カメラ映像と画像のリアルタイム合成・描画
  - **MediaRecorder API**: 映像と音声のキャプチャ・動画ファイル生成
  - **IndexedDB**: 設定状態や画像履歴データの永続化
  - **Web Workers API**: メインスレッドをブロックしない非同期の画像検証処理
  - **Screen Wake Lock API**: 録画中・使用中の画面スリープ防止
  - **Web Share API**: モバイル端末等でのネイティブ共有UIの呼び出し

## セットアップ・使い方

### セットアップ
本アプリケーションはクライアントサイド（Webブラウザの機能）のみで完結して動作するため、バックエンドサーバーの構築は不要です。

1. 本HTMLファイルをダウンロード、または任意のWebサーバー（GitHub Pages, Vercel, Netlify, ローカルサーバーなど）に配置します。
2. ブラウザのセキュリティ制限上、カメラやマイクへアクセスするためには必ず **HTTPS環境** または `localhost`（ローカル環境） で開いてください。

### 使い方

1. **アプリの起動とカメラ許可**
   - ページにアクセスすると、ブラウザがカメラ（および録画時のマイク）へのアクセスを求めてくるので「許可」してください。
   - 「AIモデルを読み込み中...」の表示後、画面にデフォルトの顔アイコンが自身の顔にオーバーレイして表示されます。

2. **顔画像の変更**
   - 画面下部の「画像（顔画像）」アイコンをタップし、好きな画像（顔写真やイラストなど）をアップロードします。
   - ※画面右上の「歯車アイコン（設定）」から「顔の自動切り抜き」をオンにしておくと、自動で顔部分だけが綺麗に抽出されます。

3. **写真撮影・動画録画**
   - **写真（カメラアイコン）**: 現在の合成画面を静止画として保存します。
   - **録画（ビデオアイコン）**: タップで録画を開始（マイク音声も収録）、再度タップで録画を停止し、動画ファイルとしてデバイスにダウンロードします。

4. **設定と履歴の活用**
   - 右上の「歯車アイコン」から設定モーダルを開くことができます。
   - **簡易表情追従**や**自動切り抜き**のON/OFF切り替えが可能です。
   - モーダル内下部には過去に使用した顔画像（直近5件）が表示され、タップするだけで瞬時に画像を切り替えることができます。
   - 別のブラウザや端末にデータを引き継ぎたい場合は、「エクスポート」で保存したファイルを新しい環境で「インポート」してください。

5. **リセット**
   - 画面下部の「リセット」ボタンを押すと、アップロードした画像が解除され、デフォルトの顔アイコンに戻ります。
